Here is how to create our repository that will hold the dependecies.

1) Pack the binary libraries in jar file
  
   jar cvf rxtx-native-files-2.1.7.jar librxtxSerial.jnilib
   
2) Deploy the files to our repository

    mvn deploy:deploy-file -Dpackaging=jar \
                           -DgroupId=org.jdesktop \
                           -Dversion=2.1.7 \
                           -DremoteRepository=repos \
                           -DrepositoryId=remote.repos \
                           -Dclassifier=mac_os_x \
                           -DartifactId=jdic-native \
                           -Dfile=jdic-native-0.9.4.jar
